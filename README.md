# Arnelio Dev Tools

Sistema de herramientas de desarrollo para proyectos Arnelio.

## Uso como Submódulo Git

Este repositorio está diseñado para ser usado como submódulo Git en otros proyectos de Arnelio.

### Instalación
```bash
git submodule add git@github.com:fernandopro/arnelio-dev-tools.git dev-tools
```

### Actualización
```bash
git submodule update --remote dev-tools
```
